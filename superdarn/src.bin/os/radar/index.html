radar

Syntax
radar --help
radar [options]
Synopsis

A dummy radar task that simulates the response of the radar hardware. Note the ROS IP address may need to be changed to 127.0.0.1 (or localhost), either in the radar site configuration file or via the control program command line options, for this task to work.

Options
--helpprint the help message and exit.
--versionprint the RST version number and exit.
-vbverbose. Log information to the console.
-lp portlisten for messages to the QNX ROS server on the local port number lport (default is 45000).
-name stiduse the restricted frequency information for the radars listed in stid which is a comma separated list of three-letter station identifier codes.
Description

Examples

Start the radar simulator for control programs to attach to.
